# SF Landmark Navigator

## Overview
A smart navigation assistant designed for tourists exploring San Francisco landmarks.

## Features
- Interactive map
- Landmark descriptions
- User-friendly interface

## Tech Stack
- JavaScript
- React
- Leaflet.js

## Usage
Clone the repository and run:
```bash
npm install
npm start
```
